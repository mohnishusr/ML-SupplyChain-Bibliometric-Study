Repository: ML-SupplyChain-Bibliometric-Study

This repository contains materials required to reproduce the bibliometric analysis in the paper:

"From Data to Decisions: A Bibliometric Review of Machine Learning Applications in Smart Supply Chain Demand Forecasting"

Contents:

wos_search_query.txt
  Exact Web of Science search query used in the study

analysis_parameters.xlsx
  Parameter settings for Bibliometrix/Biblioshiny and VOSviewer analyses

prisma_exclusion_table.xlsx
  PRISMA screening exclusion reasons and counts

data_preprocessing_notes.txt
  Data cleaning, deduplication, and harmonization procedures

bibliometrix_analysis.R
  Example script for bibliometric analysis

Note:
VOSviewer network files (.map and .net) can also be added to reproduce the visual network maps.
